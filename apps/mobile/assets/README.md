# Mobile App Assets

This folder contains the required image assets for the BerthCare mobile app.

## Required Files

You need to create the following image files:

- **icon.png** (1024x1024) - App icon
- **splash.png** (1284x2778) - Splash screen image
- **adaptive-icon.png** (1024x1024) - Android adaptive icon
- **favicon.png** (48x48) - Web favicon

## Temporary Solution

Until you have the actual images, you can either:

1. Create simple placeholder images using any image editor
2. Use online tools like https://www.figma.com or https://www.canva.com
3. Temporarily comment out the icon references in `app.json`

## Colors

Based on your app.json, the brand color is: **#1B2B65** (dark blue)
